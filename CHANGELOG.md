# Changelog

Все значимые изменения в проекте документируются здесь.

## [1.0.0] - 2026-01-20

### Добавлено
- Инициализация проекта и Git-репозитория.
- Модульная архитектура: `src/api`, `src/db`, `src/etl`.
- Поддержка Public API (города, центры, создание лидов).
- Поддержка Client API (заготовка для авторизации и профиля).
- Локальное хранилище на SQLite3.
- Скрипты синхронизации и экспорта в CSV.
- Базовая документация (README, API Guide, Backlog).
- Анализатор структуры API (генерация API_REPORT.md).
