# Go-CRM ETL

Инструментарий для автоматизированного сбора, хранения и анализа данных из Go-CRM API.

## Основные возможности
- **Синхронизация**: Получение актуальных данных о городах, центрах, лидах и клиентах.
- **Хранение**: Локальная база данных SQLite3 с простой структурой.
- **Экспорт**: Быстрая выгрузка данных в CSV для анализа в Excel/Pandas.
- **Масштабируемость**: Модульная архитектура, готовая к переходу на Supabase.

## Структура проекта
- `src/api/`: Клиенты для работы с Public и Client API.
- `src/db/`: Описание моделей данных и логика подключения к БД.
- `src/etl/`: Процессы синхронизации и экспорта.
- `scripts/`: Готовые сценарии для запуска одной командой.
- `data/`: Место хранения SQLite базы данных.
- `exports/`: Результаты выгрузки в CSV.

## Быстрый старт

### 1. Установка
```bash
# Клонирование репозитория
git clone https://github.com/riliasov/pl-gocrm-etl.git
cd pl-gocrm-etl

# Создание виртуального окружения
python3 -m venv venv
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

### 2. Настройка
Создайте файл `.env` и укажите ваши параметры доступа:
```bash
cp .env.example .env
```

### 3. Использование
- **Запуск синхронизации**: `python scripts/sync_all.py`
- **Экспорт в CSV**: `python scripts/export_data.py`
